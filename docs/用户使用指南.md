# YSClaude 用户使用指南

## 1. 首次打开

1. 安装并打开 App。
2. 如果出现邀请码页面，输入有效邀请码完成验证。
3. 邀请码会绑定当前 Android 设备，无限使用次数，更新、卸载重装可使用。

## 2. 主聊天怎么用

主页面就是聊天页。

- 在底部输入文字**回车**发送，点击发送键获取回复。
- 右侧可以快速切换模型。
- 可以发送图片和自定义表情包。
- 正在生成时，发送按钮会变成停止按钮。
- 长按聊天气泡可以编辑或删除。
- AI 消息支持语音播放。
- AI消息下面6个按键功能：编辑AI消息、删除AI消息、播放TTS、编辑用户消息、删除用户消息、重生成。（除了TTS其他功能都可以通过长按消息气泡实现，装饰作用>实际作用）。
- 点按气泡可以查看楼层。
- 气泡支持markdown语法。

主聊天窗口发送给AI的prompt构成：

- 设置页-对话设置-System Prompt
- 收藏的AI日记
- AI表情包
- 可见历史聊天上文
- 本轮运行时上下文（AI网页巡游提示、无障碍截屏提示、AI电台标记、WebView网页信息、专注事件、生理期上下文）
- 上一轮对话结束时间（最新输入前一条消息的时间戳）
- 当前时间
- 用户最新输入
- 已开启的Tool定义

顶部常用入口：

- `历史`：查看、搜索、新建、删除历史对话。
- `阅读`：进入 AI 共读，与主聊天完全独立。
- `Web`：打开WebView网页面板，打开时调用api附带当前网页内容。
- `Game`：进入多角色副本，与主聊天完全独立，支持导入酒馆格式世界书，支持多角色多api。
- `专注`：进入番茄专注，开始专注/结束专注/专注时离开APP会在**最新创建**的窗口给AI发系统提示，并调用api。
- `日历`：按日期跳转聊天记录，也可记录生理期。
- `音乐`：进入播放器和 AI 电台。
- `设置`：进入全部配置。
- 中间的 `Clawd`：预留页，未来可能接入M5Stack硬件。

聊天里还会自动带入一些上下文，比如当前时间、正在播放的音乐、专注记录、收藏日记和生理期记录。

## 3. 设置页怎么配

设置页分成这些部分：

- `API 配置`：填写 OpenAI 兼容接口的 `Base URL`、`API Key`、`Model`。
- `对话设置`：配置主聊天 System Prompt、最大输出、Prompt Cache、隐藏楼层等。
- `TTS 配置`：配置 MiniMax 语音参数，用于朗读和 AI 电台。
- `Tool 设置`：**Tool调用比较费token**。记忆库（暂时只支持我自己的记忆库接口）、Web Search（可自行获取Tavily key，每个月有免费次数）、网页读取（这个不好用，只能读到静态网页）、WebView网页交互、AI 网页巡游（自行前往UAPI获取key，AI可由此抓取指定平台热搜，访问1-3个详情页面内容后回复用户）、Native Tools（手机本地数据）、Shizuku 文件工具（需下载Shizuku工具，开启后AI可以增、改、查指定目录文件）。
- `日记`：日记总结有bug，不建议在这个页面总结，建议直接在聊天页总结，复制粘贴过来。新建、编辑、收藏、总结和上传日记。上传日记目前只支持我的记忆库接口。
- `悬浮球`：上传悬浮球素材、调整正常态/贴边态大小，开启 Android 悬浮球和相关 TTS 功能。长按悬浮球的五个功能按键：
  - 相机：需授权开启无障碍服务。短按仅截屏发给AI，AI看见手机屏幕；长按发送截屏+当前页面节点树，AI可点击、滑动、在输入框输入文字，输入文字需切换到APP自带输入法（YSClaude IME）。所有内容发送到最新创建的聊天窗口。
  - 铅笔：给AI发文字信息。发送到最新创建的聊天窗口。
  - 获取回复：调用api获取回复，AI回复内容在悬浮球上方出现，若打开TTS自动朗读。
  - 桌面歌词：快捷开启/关闭桌面歌词。
  - 返回APP

- `表情包`：管理用户表情包和 AI 表情包。
- `欢迎页`：自定义聊天页空状态中心 Logo，维护随机显示的欢迎语池，可开启系统默认欢迎语并填写名字。
- `美化`：设置聊天背景、头像、图标、字体、气泡样式和主题快照。

### 备份、恢复与诊断

在 `API 配置` 里可以：

- `创建备份并分享`
- `从备份恢复`

备份会包含数据库和自定义素材，恢复后需要重启 App。

数据库诊断可以查看修正空楼层。

## 4. AI 共读

进入 `阅读`：

1. 在 `书架` 导入 `txt` 或 `epub`。
2. 打开书籍后，按章节阅读。
3. 长按句子可以划线或取消划线。
4. 右侧共读面板可以直接和 AI 讨论。
5. 在 `总结` 里可以看这本书的划线、AI 总结和读书心得。
6. 在 `设置` 里可以单独配置共读 API 和总结提示词。

## 5. 音乐与 AI 电台

进入 `音乐`：

- 可播放歌曲、暂停、切歌、拖动进度。
- 支持歌词滚动，点击歌词可跳转。
- 可以开桌面歌词。
- `歌单管理` 里可以填写网易云 API 地址，扫码登录后导入歌单。
- AI 电台会基于当前歌单生成节目、主持词和总结。

## 6. 番茄专注

进入 `专注`：

- 新建今天的任务。
- 开始计时后可暂停、继续、完成或放弃。
- 支持手动补记一次专注。
- 统计页可以按日期查看记录和任务分布。

## 7. 多角色副本

进入 `Game`：

1. 新建一个副本。
2. 配置副本名称、简介、剧本和基础设定。
3. 给旁白、总结 AI、角色分别绑定 API 配置。
4. 进入房间后，先发言，再手动选择让谁回复。

副本支持：

- 角色头像、用户头像、牌面图片
- 消息编辑和删除
- 清空房间
- 隐藏指定楼层范围

## 8. 常见权限

部分功能需要额外权限或外部服务：

- 悬浮球需要系统悬浮窗权限。
- 日历功能需要系统日历权限。
- 应用使用统计需要系统使用情况访问权限。
- Shizuku 文件工具需要先安装并运行 Shizuku。
- 音乐歌单导入需要可用的网易云 API。
- 语音播放和 AI 电台需要正确配置 TTS。
